### 1. My Diary（個人制作）

#### ■ 概要

本作品は、日々の行動や感情を言語化し、自己理解を深めることを目的とした内省・振り返り用の日記帳Webアプリです。  
ユーザーはその日の出来事、感じたこと、学びを入力し、時系列で振り返ることができます。  
また、テーマ変更機能も実装しています。

#### ■ 使用技術

* Visual Studio Code
* HTML / CSS / JavaScript

#### ■ 制作期間

約2か月

#### ■ 工夫した点

* 入力内容を整理しやすいUI設計を意識し、「今日の気分」や「日記」など、内容ごとに分けて記録できる構成にしました
* テーマ変更機能を実装し、ユーザーが継続して使いたくなるデザイン性を意識しました

#### ■ 学んだこと

* フロントエンド開発におけるUI/UX設計の重要性
* JavaScriptを用いた状態管理と画面更新処理
